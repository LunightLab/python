# 이미지 처리 K-means clustering 

## Base install
```zsh
pip3 install flask numpy opencv-python-headless scikit-learn matplotlib pillow
```

## File tree
```zsh
your_project/
│
├── k-means.py
```

## Run
```zsh
python3 k-means.py
```

- http://127.0.0.1:5000로 접속
